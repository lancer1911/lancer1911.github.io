Three focused tools.

Each project is built around a different audio workflow: live speech, prepared recordings, and speech synthesis.

🎙️

ASR-Live

Fully offline real-time multilingual speech recognition for Apple Silicon. It captures live audio, transcribes speech, applies LLM-based semantic correction, and can produce multilingual translations without cloud calls.

Real-time ASR Whisper / SenseVoice LLM correction Local privacy
Open repository
📝

ASR-Offline

A local audio-file transcription and subtitle editing tool for Apple Silicon Macs. It processes common audio formats, performs ASR, correction, translation, diarization, synchronized playback, and exports practical transcript formats.

Audio files Subtitle editing Diarization TXT / SRT / JSON / MD
Open repository
🔊

TTS-Offline

Fully offline text-to-speech synthesis for Apple Silicon. It converts text into natural speech using local models, with multi-voice dialogue support, per-sentence emotion control, audiobook-oriented production, and voice cloning from reference audio.

Text to speech Multi-voice Emotion control Voice cloning
